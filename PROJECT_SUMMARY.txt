========================================
  快速開始步驟
========================================

1️⃣ 安裝套件
   npm install

2️⃣ 設定環境變數（編輯 .env）
   - LINE_CHANNEL_ACCESS_TOKEN
   - LINE_CHANNEL_SECRET
   - GOOGLE_SHEET_ID (選用)

3️⃣ 啟動伺服器
   npm start

4️⃣ 使用 ngrok 建立公開網址
   ngrok http 3000

5️⃣ 在 LINE Developers Console 設定 Webhook URL

6️⃣ 掃描 QR Code 加入 Bot 測試
